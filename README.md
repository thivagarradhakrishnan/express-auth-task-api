# Task Auth API (Node.js + MongoDB)

A simple task manager app with JWT-based login/register using phone number.

## Features
- Register/Login via Phone Number
- Create, List, and Update Tasks
- JWT Auth Middleware
- Clean Folder Structure (MVC)
